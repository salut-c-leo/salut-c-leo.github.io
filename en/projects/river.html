Take Me To The River
Description and project breakdown.

  • Description
  • Project breakdown

Description

  • Solo
  • 2 weeks
  • Source SDK 2013

In this level made to be part of Half-Life 2: Episode 2, Gordon has to cross the river via a bridge controlled by the Combine.

Project breakdown

I gave myself two weeks to make this level, to not overscope, and not spend too much time on it. I was capable of it when I made levels for Map Labs, so I wanted to get back in those constraints.

Level Design Document (in French)

Where I could improve

One of the difficulties in this project is that the final combat section had to be cut, due to bugs in the Source Engine preventing NPCs of shooting RPGs, bugs I couldn't fix on my own, and iterating on it to find a better solution didn't end up being successful. Making a test map would have probably helped identifying those bugs and planning the level accordingly.
